# 👟 Shoe Webpage

A modern and responsive **multi-page shoe website** built using **HTML5** and **CSS3**. This repository currently contains the **Landing Page**, featuring a clean and premium UI with a responsive design.

## ✨ Features

- Modern and responsive layout
- Clean navigation bar
- Hero section with featured shoe
- Search bar and shopping cart UI
- Customer statistics section
- "Shop Now" call-to-action
- Multi-page website structure

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome
